# Mohammad Ayaan Siddiqui's Portfolio Website 🚀

## Description 📝

This is the portfolio website for Mohammad Ayaan Siddiqui, a self-taught full-stack blockchain developer, technical content writer, and web3 enthusiast. 💻✍️ The website showcases Ayaan's skills, projects, and experiences in the world of blockchain and web development. 🌐💡 It serves as a platform to connect with potential clients, collaborators, and like-minded individuals in the industry. 🤝🌟

## Features ✨

- 🎨 Sleek and modern design with a dark mode option
- 🌐 Fully responsive and mobile-friendly layout
- 🚀 Fast and smooth navigation with optimized performance
- 💼 Detailed sections for skills, projects, and work experience
- 🎯 Interactive hover effects and animations to engage visitors
- 📞 Easy ways to connect and get in touch with Ayaan
- 🌍 SEO-friendly structure and metadata for better visibility
- 🛠️ Built with the latest web technologies and best practices

## Tech Stack 🛠️

- ⚛️ Next.js - React framework for server-side rendering and static site generation
- 🎨 Tailwind CSS - Utility-first CSS framework for rapid UI development
- 🚀 TypeScript - Static typing for enhanced code quality and developer experience
- 🌐 Vercel - Cloud platform for seamless deployment and hosting
- 🎥 Canvas Confetti - Lightweight library for adding confetti animations
- 🖼️ Next Image - Optimized image component for better performance
- 📱 Responsive Design - Ensuring a seamless experience across devices
- 🌙 Dark Mode - Providing a visually appealing option for low-light environments
- 🎨 Engaging Animations - Enhancing user experience with subtle animations
- 📝 Markdown - Simplifying content creation and management
- 🔍 SEO Optimization - Improving visibility and search engine rankings
